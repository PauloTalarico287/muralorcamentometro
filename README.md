# Orcamentometro
Extração e análise de dados da execução orçamentária da prefeitura de SP

O Orçamentômetro é um projeto de jornalismo de dados, criado pela Agência Mural de Jornalismo das Periferias, que monitora a execução orçamentária da Prefeitura de São Paulo, com foco em transparência, desigualdades territoriais e acesso à informação para moradores das periferias da cidade. Os dados são utilizados para uma página na Agência Mural (https://orcamentometro.agenciamural.org.br/)). 

O projeto transforma dados públicos complexos em visualizações acessíveis, permitindo acompanhar quanto do orçamento municipal foi prometido e efetivamente executado ao longo do ano.

🎯 Objetivos

Acompanhar a execução do orçamento municipal de São Paulo
Comparar promessas orçamentárias x execução real
Evidenciar desigualdades regionais entre subprefeituras
Facilitar o acesso da população a dados públicos

Apoiar reportagens, investigações e controle social

🗺️ O que o Orçamentômetro mostra

Execução orçamentária por:

Secretarias municipais
Subprefeituras
Fundos municipais
Emendas parlamentares

Evolução dos gastos ao longo do tempo

Mapas temáticos com recorte territorial

Gráficos e indicadores de execução (%)

📊 Fontes de dados

O projeto utiliza dados públicos oficiais, extraídos do portal da Prefeitura de São Paulo, em especial a página sobre Execução orçamentária.

⚠️ Os dados são atualizados conforme a disponibilidade dos portais oficiais.



📰 Aplicações jornalísticas

O Orçamentômetro foi desenvolvido para:
Produção de reportagens baseadas em dados
Apoiar investigações sobre gastos públicos
Visualizar desigualdades no investimento público
Fortalecer o debate público e o controle social
